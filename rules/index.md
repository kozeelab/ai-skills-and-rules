## AI 使用指南

1. **优先查看本索引**：AI 在处理任务前，应首先查看本索引文件定位需要的规则
2. **核心规则必读**：所有任务都应遵循 [AI 协作开发规范](./common/rule.md)，其中包含各规范文件的交叉引用
3. **语言规范按需加载**：根据项目语言加载对应的语言特定规则

### 📝 索引自动维护指令

> **强制要求**：AI 在对 `rules/` 目录进行文件增删改时，**必须同步更新本索引**。

- **新增规则**：在「快速导航」和「详细索引」中添加对应条目 + 更新日期
- **删除规则**：从「快速导航」和「详细索引」中移除对应条目 + 更新日期
- **修改规则**：更新受影响的条目描述和日期
- 文件名使用小写+中划线格式，规则文件使用 Markdown 格式
- 变更历史由 git log 追踪，索引中不再单独记录
