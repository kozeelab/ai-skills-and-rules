# 项目总结 Skill

## 功能说明

此Skill用于总结项目中的亮点和难点，生成一段适合直接复制到简历上的项目经历。

## 使用方法

1. 分析项目结构和代码
2. 识别项目的技术亮点和挑战
3. 生成结构化的项目经历描述
4. 输出适合简历的专业语言

## 执行步骤

1. **项目分析**
   - 扫描项目目录结构
   - 识别使用的技术栈和框架
   - 分析项目的核心功能和模块

2. **亮点提取**
   - 技术创新点
   - 性能优化措施
   - 架构设计优势
   - 解决的关键问题

3. **难点分析**
   - 技术挑战
   - 解决方案
   - 克服的困难

4. **简历输出**
   - 结构化的项目描述
   - 量化的成果指标
   - 专业的技术术语
   - 清晰的职责描述

## 输出模板

```
项目名称：[项目名称]

项目描述：[简要描述项目的目标和功能]

个人职责：
- [职责1：具体描述负责的工作内容]
- [职责2：具体描述负责的工作内容]
- [职责3：具体描述负责的工作内容]

技术亮点：
- [亮点1：技术创新或优化]
- [亮点2：技术创新或优化]
- [亮点3：技术创新或优化]

解决的难点：
- [难点1：挑战及解决方案]
- [难点2：挑战及解决方案]
- [难点3：挑战及解决方案]

项目成果：
- [成果1：量化的成果]
- [成果2：量化的成果]
- [成果3：量化的成果]
```
